# Smart Login Form

A simple smart login form with real-time validation built using HTML, CSS, and JavaScript.

## 🚀 Features

- Real-time input validation
- Error and success messages
- Password validation
- Phone number validation (+234 format)
- Form reset after successful submission

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📌 What I Learned

- DOM manipulation
- Event handling
- Form validation
- UI/UX improvements
- Project structuring
